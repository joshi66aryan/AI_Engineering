## Project Overview

This project demonstrates basic **data analysis, data manipulation, and visualization** using the Titanic dataset with **Pandas** and **Matplotlib**.

The goal is to understand the dataset structure, clean and manipulate data, and visualize important patterns.

---

## Dataset Information

- Rows: **891 passengers**
- Columns: **12 features**

---

## What I Did in This Project

### Data Exploration

- Checked dataset shape, columns, and data types
- Identified missing values in:
  - Age
  - Cabin
  - Embarked

---

### Data Cleaning & Manipulation

- Filled missing `Age` values using the mean
- Selected useful columns for analysis
- Filtered data using **`loc` and `iloc`**
- Created multiple cleaned and filtered DataFrames

---

### Data Visualization

We visualized the data using **Matplotlib**:

- **Bar Graph**

- Survival count
- Survival rate by gender

- **Pie Chart**

- Gender distribution

- **Histogram**

- Age distribution
- Fare distribution

- **Scatter Plot**

- Relationship between Age and Fare
